Syntax to push code
```bash
git add .           # Staging phase
git commit -m <"Your message"> # Commit Phase
git push origin <Branch Name>        # Push Code
```

EG
```bash
git add .
git commit -m "Todays work"
git push origin main
```

Syntax to pull code
```bash
git pull origin <Branch Name>        # Push Code
```

EG
```bash
git push origin main
```
